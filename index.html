سوينا تغيير
كيف نكسل التغيير؟ git restore
كيف نشيل ملف من المسرح بعد ما إضافته؟ git reset
كيف نضيفه للمسرح؟ git add
كيف نسوي نسخة؟ git commit -m "<اكتب اللي تبيه>"
كيف نعرض كل النسخ اللي سويناها؟ git log
كيف اعرف الفروع اللي عندي؟ git branch
كيف نسوي برانش جديد؟ git branch
كيف نروح لفرع؟ git checkout كيف نرفع؟ git push
